## TBU-TS

### 运行项目
```bash
	yarn install //下载依赖
	yarn serve  //运行项目
```
### 项目结构
```bash
		-node_modules  //项目依赖
		-public		//公共资源
		-src		//源代码
				-api		//请求文件
					http.ts  //封装请求的文件
					requestList.ts		//请求的ts
				-pages		//页面文件
				-common		//公共资源
					-ts		//ts文件
					-scss		//scss文件
				-components		//组件文件
				-static		//静态资源文件
				-store		//vuex文件
		package.json		//页面配置
```
